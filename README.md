Globale Einstellungen AddOn für REDAXO 4
========================================

Mit diesem Addon kann man globale MetaInfos setzen die für die gesamte Website gültig sind.

Hinweise
--------

* Getestet mit REDAXO 4.6
* AddOn-Ordner lautet: `global_settings`
